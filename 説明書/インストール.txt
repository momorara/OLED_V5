2024/03/05　インストール手順

2024/03/07　Bullseye 10.13で確認
2024/03/05　BookWorm 12.5で確認
2024/03/05　Bullseye 11.9で確認

I2cを有効化
sudo raspi-config
にてi2cを有効にしてください。

基板を装着した状態で
i2cdetect -y 1
とすると、
OLED 3c
AHT  38
と表示されます。

ライブラリをインストール

Buster、Bullseyeの場合
pip3 install luma.core
pip3 install luma.oled
pip3 install adafruit-circuitpython-ahtx0

BookWormの場合
pip install --break-system-packages luma.core
pip install --break-system-packages luma.oled
pip install --break-system-packages adafruit-circuitpython-ahtx0

外部フォント関係のインストール
sudo apt install python3-pil # 標準でインストール済みかも
sudo apt-get install fonts-dejavu
sudo apt-get install fonts-ipafont


プログラムをダウンロード
git clone https://github.com/momorara/OLED_V5

プログラムのディレクトリ OLED_V5 へ移動
cd OLED_V5
として、ls
python3 xxx.py としてサンプルプログラムを試してみてください。
